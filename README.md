# EXERCISE 9 - SHOUTOUTS TO EVERYONE
## Problem Statement
Write a program to pronounce list of names using win32 API (pywin32 library). If you are given a list l as follows:
<br>
    l = ["Ali", "Ahmed", "Arman"]
<br>
    Your program should pronouce:
<br>
        Shoutout to Ali
<br>
        Shoutout to Ahmed
<br>
        Shoutout to Arman
